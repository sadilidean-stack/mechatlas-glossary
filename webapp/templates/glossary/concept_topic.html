Open-Loop vs. Closed-Loop Control

Definition

Open-loop control system: Control signal is determined solely by the input or reference signal, without monitoring the actual output.

Closed-loop control system (feedback control): The system continuously monitors the output, compares it with the desired setpoint, and adjusts inputs to reduce the error — using feedback.

Function

Type Strengths Weaknesses / Limitations
Open-loop Simple, inexpensive, easy to implement — works if behavior predictable and environment stable. No correction for disturbances or change in system — output may drift from desired.
Closed-loop (feedback) Can correct for disturbances, adapt to changes, maintain stability, and improve accuracy. More complex, more expensive; potential stability issues (oscillation, overshoot) if not properly designed.

5/2 Directional Control Valve Operation

Definition

A valve with 5 ports and 2 positions, commonly used in pneumatic systems. Typically: one supply port (P), two actuator ports (for the two sides of a double‑acting cylinder), and two exhaust ports (one for each actuator port) for exhaust/venting.

Function

Switches between two positions to alternate which side of the actuator receives supply air (or fluid) and which side is exhausted. In one position, supply flows to port A and port B is vented; in the other, supply flows to port B and port A is vented. This enables extend and retract motion of a double‑acting cylinder.
