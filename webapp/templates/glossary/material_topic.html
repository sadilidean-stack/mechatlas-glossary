An electronic module or circuit that processes, amplifies, filters, linearizes, or converts the raw output signal from a sensor into a stable, standardized signal suitable for controllers or data acquisition systems.

Function

Ensures that sensor outputs (which may be weak, noisy, or non-standard) are converted into reliable, usable signals for measurement, control, or monitoring; improves signal quality, reduces noise vulnerability, and allows long-distance transmission.

Stepper Driver Module

Definition

An electronic circuit/module that receives control signals (pulses + direction) and energizes the stepper motor windings accordingly, controlling current and coil activation.

Function

Translates control pulses into the correct sequence of currents to the motor coils, enabling the stepper motor to turn step-by-step; also regulates current/voltage to protect the motor and ensure stable operation.

Stepper Motor

Definition

An electromechanical motor that converts discrete electrical pulses into precise, incremental angular shaft rotations.

Function

Provides controlled mechanical output: each input pulse rotates the shaft by a fixed “step,” enabling accurate positioning, repeatable motion, and good torque at low speeds or when stopped.

3/2 Directional Control Valve

Definition

A valve with 3 ports (inlet, outlet/actuator, exhaust/return) and 2 positions.

Function

Used to control a single‑acting actuator or simple flow path. In one position it supplies fluid/air to the actuator, in the other it vents/exhausts the actuator allowing it to return or stop.

4/2 Directional Control Valve

Definition

A valve with 4 ports and 2 positions, typically used with double‑acting actuators or systems requiring reversing fluid/air flow.

Function

By switching positions, it connects supply to one actuator port and exhausts the other, enabling the actuator to extend or retract (forward/reverse motion).

4/3 Directional Control Valve

Definition

A valve with 4 ports and 3 positions (commonly a 4‑way valve with a center/neutral position) used in hydraulic or pneumatic circuits.

Function

Allows extend, retract, or neutral/fluid‑blocked states for a double‑acting actuator enabling motion in either direction or holding the actuator stationary (locked or vented) depending on the center position design.

5/2 Directional Control Valve

Definition

A valve with 5 ports and 2 positions, commonly used in pneumatic systems controlling double‑acting cylinders.

Function

Directs supply air (or fluid) alternately to one side of a double‑acting actuator while exhausting the other, enabling extend in one position and retract in the other. The extra port(s) allow separate exhaust paths for better control.

7400 IC (NAND)

Definition

7400 is a 7400-series TTL IC containing four independent 2-input NAND gates in a single 14-pin DIP package.

Function

Each gate outputs LOW only when both inputs are HIGH; otherwise, the output is HIGH.

7402 IC (NOR)

Definition

7402 is a 7400-series TTL IC containing four independent 2-input NOR gates in a single 14-pin DIP package.

Function

Each gate outputs HIGH only when both inputs are LOW; otherwise, the output is LOW.

7404 IC (NOT)

Definition

7404 is a 7400-series TTL IC containing six independent NOT gates (inverters) in a single 14-pin DIP package.

Function

Each gate outputs the logical inverse of its input: HIGH becomes LOW, LOW becomes HIGH.

Definition

7408 is a standard 7400-series TTL IC containing four independent 2-input AND gates in a single 14-pin DIP package.

Function
